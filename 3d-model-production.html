実物3Dモデル制作・登録

ARで「実物と相違がない」品質を目指すための制作管理ページです。ここでは架空の料理モデルは使わず、施設の実メニューを3DスキャンしたGLBだけを登録対象にします。

まずは使用頻度の高い10品を本物3D化します。未登録の料理はARで「実物3Dモデル未登録」と表示されます。

制作手順

1. 実物を用意

実際に提供する器・盛り付け・量で撮影します。

2. 3Dスキャン

Polycam、Luma AI、Scaniverse等でGLBを書き出します。

3. 軽量化

スマホAR用に1品あたり10MB以下を目安にします。

4. 登録

assets/models/ に置き、real_models.js に登録します。

優先10品の登録状況

登録例

GLBファイルを assets/models/shoyu_ramen.glb に置いた場合は、real_models.js に次のように追加します。

window.registerRealFoodModel('shoyu_ramen', {
  url: 'assets/models/shoyu_ramen.glb',
  scale: '.9 .9 .9',
  rotation: '0 0 0',
  position: '0 .12 0'
});
